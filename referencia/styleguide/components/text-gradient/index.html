Text Gradient

Animated and static gradient text effects using CSS utility classes. Built with CSS variables for easy color customization.

<span className="text-gradient-animated">highlighted text</span>

Animated Gradient

Continuously shifting gradient that cycles through purple, blue, cyan, and lilac. Duration: 5s, ease-in-out, infinite loop.

Partial Text (inline span)

Apply the gradient to only part of a heading — the rest stays white. This is the pattern used in the landing page hero.

Gradient Tokens

The gradient colors are defined as CSS custom properties in globals.css. Override them to change the palette everywhere.

Purple

--gradient-1

#7C3AED

Blue

--gradient-2

#3B82F6

Cyan

--gradient-3

#06B6D4

Lilac

--gradient-4

#A78BFA

:root {
  --gradient-1: #7c3aed;
  --gradient-2: #3b82f6;
  --gradient-3: #06b6d4;
  --gradient-4: #a78bfa;
}

Notes

  • Uses both -webkit-background-clip and background-clip for Safari and Chrome compatibility.
  • Uses -webkit-text-fill-color: transparent for reliable cross-browser rendering.
  • The animation uses background-position shift, which is GPU-composited and performant.
  • Avoid applying to very long paragraphs — the effect works best on short highlighted phrases or headings.
  • The gradient background-size: 300% 100% creates the scrolling illusion by making the gradient wider than the text.
