Accessibility

  • Card uses semantic div elements with data-slot attributes
  • Use proper heading hierarchy within cards (h2, h3, etc.)
  • Interactive cards should use role="article" or wrap in a link
  • Ensure sufficient color contrast for card text on card background
  • Focus management handled by interactive children (buttons, links)
