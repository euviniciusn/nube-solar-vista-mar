Accessibility

  • Built on Radix UI Progress primitive with role="progressbar"
  • Automatically sets aria-valuenow, aria-valuemin, and aria-valuemax
  • Pass value={null} for indeterminate state
  • Use getValueLabel to provide a custom accessible label
  • Screen readers announce progress changes automatically
