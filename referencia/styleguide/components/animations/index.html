Animations

Premium scroll-triggered animations powered by Framer Motion. 7 components with built-in prefers-reduced-motion support.

<FadeIn>
  <Card>...</Card>
</FadeIn>

FadeIn

Fade in with directional offset. Supports up (default), down, left, right.

Default (up)

Fades in from 24px below with 0.6s duration.

From left (200ms delay)

Slides in from 40px left.

From right (400ms delay)

Slides in from 40px right.

<FadeIn direction="up" delay={0} duration={0.6}>
  <Card>Default fade up</Card>
</FadeIn>

<FadeIn direction="left" delay={0.2}>
  <Card>Slide from left</Card>
</FadeIn>

PriceCounter

3-phase price animation: displays a market value, pauses, then counts down to a discounted price. Shows strikethrough original, savings badge, and reajuste line. Uses requestAnimationFrame.

Valor de mercado

R$5.100

/mês

import { PriceCounter } from "@/components/animations"

<PriceCounter
  startValue={5100}   // Market value (shown first)
  finalValue={4297}   // Discounted price (after countdown)
  pauseDuration={2000} // ms to pause before falling
  fallDuration={2000}  // ms for the countdown animation
/>

{/* Phases:
 * 1. Shows startValue with "Valor de mercado" label (pauseDuration)
 * 2. Counts down to finalValue with pulse + color transition
 * 3. Shows gradient text, strikethrough original, savings badge
 *
 * Mobile: shorter timings (1.5s), smaller text
 * Respects prefers-reduced-motion (instant)
 */}

Notes

  • All components use useReducedMotion from Framer Motion
  • When reduced motion is preferred, animations show instant opacity fade only
  • Viewport trigger uses once: true — animations fire once and don't replay
  • Custom easing: cubic-bezier(0.21, 0.47, 0.32, 0.98)
  • All delay/duration props use seconds (not milliseconds)
  • CountUp uses requestAnimationFrame for smooth 60fps counting
  • TextReveal splits text by spaces; each word is an inline-block motion.span
