Hover States

Premium micro-interactions that add depth and polish. All hover effects use cubic-bezier(0.21, 0.47, 0.32, 0.98) for a natural, springy feel.

Card Hover — Glow Lift

Card with glow variant lifts 4px on hover with enhanced box-shadow. 0.4s cubic-bezier transition.

Hover me

Card lifts 4px, glow intensifies, icon scales 1.1x and rotates 3deg.

Glow Pulse

Continuous pulsing glow animation (3s infinite). Used on IA card and price card.

.card-glow {
  transition: all 0.4s cubic-bezier(0.21, 0.47, 0.32, 0.98);
}
.card-glow:hover {
  box-shadow: 0 0 30px var(--glow-primary-hover), ...;
  transform: translateY(-4px);
}

/* Pulsing glow for special cards */
.glow-pulse {
  animation: glow-pulse 3s ease-in-out infinite;
}

Button Hover — Scale

Buttons scale 1.02x on hover and 0.98x on active press for a tactile feel.

button:not(:disabled):hover {
  transform: scale(1.02);
}
button:not(:disabled):active {
  transform: scale(0.98);
}

/* Applied globally via globals.css */

Reduced Motion

  • All transitions/animations are disabled when prefers-reduced-motion: reduce is active
  • Framer Motion components use useReducedMotion() hook
  • CSS animations reduced via global @media rule
  • Gradient text animation stops (single frame, no movement)
  • Smooth scroll reverts to instant scroll
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
  html { scroll-behavior: auto; }
  .text-gradient-animated {
    animation: none;
    background-size: 100% 100%;
  }
}
