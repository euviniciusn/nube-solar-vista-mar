Accessibility

  • Uses native table elements for full semantic meaning
  • TableCaption provides accessible description for the table
  • Supports data-state="selected" for row selection styling
  • Horizontal scroll container prevents layout break on small screens
  • Use scope="col" on headers for complex tables
