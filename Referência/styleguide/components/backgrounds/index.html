Backgrounds

Texture and depth utility classes for section backgrounds. Add visual interest and eliminate the flat, generic look.

<section className="bg-grid bg-muted/50">...</section>
<section className="bg-noise bg-radial-glow relative">...</section>

bg-grid

Subtle grid pattern using primary color at 3% opacity. 64px cell size.

64px grid pattern

.bg-grid {
  background-image:
    linear-gradient(rgba(124, 58, 237, 0.03) 1px, transparent 1px),
    linear-gradient(90deg, rgba(124, 58, 237, 0.03) 1px, transparent 1px);
  background-size: 64px 64px;
}

<section className="bg-grid bg-muted/50">...</section>

bg-noise

Fractal noise overlay via SVG filter. Uses ::before pseudo-element at z-index 1. Content needs z-index 2+.

Subtle grain texture (look closely)

.bg-noise {
  position: relative;
}
.bg-noise::before {
  content: '';
  position: absolute;
  inset: 0;
  background-image: url("data:image/svg+xml,...");
  pointer-events: none;
  z-index: 1;
}

/* Content must be above the noise layer */
<section className="bg-noise relative">
  <div className="relative z-[2]">...</div>
</section>

bg-radial-glow

Radial gradient glow from top center. 8% primary opacity, fades to transparent at 60%.

Soft radial glow from top

.bg-radial-glow {
  background-image: radial-gradient(
    ellipse at 50% 0%,
    rgba(124, 58, 237, 0.08) 0%,
    transparent 60%
  );
}

bg-radial-glow-intense

More intense version centered in the section. 12% opacity, fades at 50%. Use for hero/CTA sections.

Intense centered glow

.bg-radial-glow-intense {
  background-image: radial-gradient(
    ellipse at 50% 50%,
    rgba(124, 58, 237, 0.12) 0%,
    transparent 50%
  );
}

Combined

Multiple background classes can be combined for richer effects.

bg-noise + bg-grid + bg-radial-glow

<section className="bg-noise bg-grid bg-radial-glow relative">
  <div className="relative z-[2]">
    Content here
  </div>
</section>

Usage in Landing Page

  • Hero: bg-noise bg-radial-glow
  • Problema: bg-grid
  • Serviços: bg-grid
  • IA: bg-radial-glow-intense
  • Equipamentos: bg-grid
  • Manutenção: bg-radial-glow
  • Investimento: bg-radial-glow-intense
  • Diferenciais: bg-grid
  • CTA: bg-noise bg-radial-glow
  • Alternate textured/clean sections for visual rhythm
