Hover Effect

Add icon-hover class for scale + rotation on group hover.

Hover this card

Icon scales 1.1x and rotates 2deg

.icon-hover {
  transition: transform 0.3s ease;
}
.group:hover .icon-hover {
  transform: scale(1.1) rotate(2deg);
}

<div className="group">
  <Icon className="icon-hover" />
</div>

Notes

  • All icons use weight="duotone" for the two-tone visual style
  • Duotone renders a secondary layer at 20% opacity of the main color
  • Uses currentColor so Tailwind text color classes work
  • Package: @phosphor-icons/react
  • Icons are tree-shakeable, only imported icons are bundled
