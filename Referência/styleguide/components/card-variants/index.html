Card Variants

Cards with depth, glow, glass, and premium effects. Four variants beyond the default for different contexts.

import { Card } from "@/components/ui/card"

<Card variant="glow">...</Card>
<Card variant="glass">...</Card>
<Card variant="bordered">...</Card>

{/* Premium uses CSS class instead of variant prop */}
<div className="card-premium bg-noise">...</div>

Default

The standard card with subtle border and shadow. Backwards compatible with all existing usage.

Default Card
Standard border and shadow-sm.

This is the original card style with no variant prop.

<Card>
  <CardHeader>
    <CardTitle>Default Card</CardTitle>
  </CardHeader>
</Card>

Variant: "glow"

Primary purple glow with hover lift effect. Used for feature cards, service cards, and interactive content.

Glow Card
Hover me for the lift effect.

Subtle purple glow border with animated hover that lifts the card and intensifies the glow.

Another Glow
Same variant, consistent feel.

All glow cards share the same transition timing and shadow intensity for visual consistency.

<Card variant="glow" className="group">
  <CardHeader>
    <CardTitle>Glow Card</CardTitle>
  </CardHeader>
  <CardContent>...</CardContent>
</Card>

Variant: "glass"

Frosted glass effect with backdrop blur. Best used over colored or gradient backgrounds.

Glass Card
Frosted glass over gradient.

Uses backdrop-filter: blur(12px) with semi-transparent background. Works best on colored sections.

<Card variant="glass">
  <CardHeader>
    <CardTitle>Glass Card</CardTitle>
  </CardHeader>
  <CardContent>...</CardContent>
</Card>

Variant: "bordered"

Clean card with emphasized border. Hover changes border to primary color. Best for tables and data displays.

Bordered Card
Hover to see border highlight.

Thicker 1.5px border with hover transition to primary color. Clean and data-focused.

<Card variant="bordered">
  <CardHeader>
    <CardTitle>Bordered Card</CardTitle>
  </CardHeader>
  <CardContent>...</CardContent>
</Card>

CSS Class: "card-premium"

Animated rotating gradient border with multi-layer glow. Used for hero pricing cards and premium showcase elements. Uses @property for smooth CSS angle animation.

{/* Premium card uses CSS class, not a Card variant */}
<div className="card-premium bg-noise">
  <div className="relative z-[2] flex flex-col items-center rounded-[1.5rem] p-10 md:p-14">
    {/* Content here */}
    <span className="text-gradient-animated">R$ 5.100</span>
    <div className="separator-gradient my-8 w-full" />
  </div>
</div>

{/* Key CSS classes:
 * .card-premium — rotating gradient border + glow
 * .bg-noise — subtle texture overlay
 * .text-gradient-animated — animated gradient text
 * .separator-gradient — gradient horizontal rule
 */}
