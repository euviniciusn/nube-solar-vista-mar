Badge

Displays a badge or a component that looks like a badge.

import { Badge } from "@/components/ui/badge"

All available badge variants.

As Child (Link)

Render badge as a link using asChild.

Clickable BadgeOutline Link
<Badge asChild>
  <a href="#">Clickable Badge</a>
</Badge>

Props

PropTypeDefault
variant"default" | "secondary" | "destructive" | "outline" | "ghost" | "link""default"
asChildbooleanfalse

Accessibility

  • Uses span element by default (non-interactive)
  • Use asChild with an anchor or button for interactive badges
  • Color should not be the only way to convey meaning - use icons or text
  • Ensure sufficient contrast between badge background and text
